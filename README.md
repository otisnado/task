# Task CLI

Just a CLI application that help you to store task in a MongoDB Database

Requeriments:
- MongoDB Database
- Config file in $HOME/conftask/app.env (must exists before run app)
- Task bin CLI app
- Source code (If you want to build by yourself)

Config file sample:

````
DB_NAME=database
DB_HOST=localhost
DB_USER=user
DB_PASSWORD=password
````
